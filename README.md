# Retail Data Analysis Project

This project contains retail and e-commerce data alongside a master data analysis prompt and its corresponding Jupyter Notebook. It demonstrates an automated, beginner-friendly approach to data analysis using Python and Pandas.

## Files Included

- **`product_master.csv`**: Contains details about the product catalog, including `product_id`, category, brand, unit price, and cost.
- **`sales_transactions.csv`**: Contains individual purchase records, mapping customers to the products they bought, alongside purchase dates, regions, and discount percentages.
- **`Data_Analysis_Master_Prompt.md`**: A system prompt designed to dynamically analyze any given CSV structured datasets, perform automated cleaning, apply feature engineering, and output a Jupyter Notebook.
- **`Sales_Product_Analysis.ipynb`**: The resulting Jupyter Notebook generated from the `Data_Analysis_Master_Prompt.md`. It performs a complete end-to-end data analysis on the provided sales and product CSVs, complete with Key Performance Indicators (KPIs) and data visualizations.

## How to View the Analysis
You can open `Sales_Product_Analysis.ipynb` using Jupyter Notebook, JupyterLab, or any compatible IDE (such as VS Code) to view the step-by-step data cleaning, merging, KPI calculations, and charts.

## Technologies Used
- Python 3
- Pandas (Data Manipulation)
- Matplotlib & Seaborn (Data Visualization)
- Jupyter Notebook
